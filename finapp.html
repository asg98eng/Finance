☁️ Sincronización con Google Drive

La app guarda un archivo finapp_data.json en tu Google Drive para sincronizar entre PC y móvil.

  1. Ve a console.cloud.google.com
  2. Crea un proyecto → activa Google Drive API
  3. Crea credenciales OAuth 2.0 → Aplicación web
  4. En "Orígenes autorizados" añade la URL donde abres este archivo
  5. Copia el Client ID aquí abajo
Alternativa sin configuración: Exporta el JSON y guárdalo manualmente en Google Drive. En el otro dispositivo, importa ese mismo archivo.
