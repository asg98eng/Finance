Introduce tu Client ID de Google Cloud para activar la sincronización automática entre dispositivos.

  1. Ve a console.cloud.google.com
  2. Crea proyecto → APIs → Google Drive API → Habilitar

  4. Origen JS autorizado: https://asg98eng.github.io
  5. Copia el Client ID generado
